Esse projeto simula um gerenciador de prontuários de um consultório de pediatría
armazenando os dados da criança.

Cada prontuário tem um número referente ao mesmo. Esse número está relacionado
a quantos já foram criados no sistema, por isso, ele não é alterado mesmo que
algum itém seja removido ou atualizado.

São usadas as bibliotecas: matplotlib.pyplot e numpy. Caso não se tenha alguma 
dessas bibliotecas é necessários baixa-las para que o código execute sem problemas.

O código esta dividido em 2 arquivos: um com o código principal main.py e outro com 
o arquivo com as funções para exibir os gráficos graficos.py.

Há também o arquivo de entrada e saída com alguns prontuários já cadastrados.
Esse arquivo armazena as informações de cada prontuário por linha, sendo a primeira 
linha reservada para o número do ultimo prontuário cadastrado.
Esse arquivo precisa estar na mesma pasta que o arquivo main.py
